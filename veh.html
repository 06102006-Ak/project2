Smart Parking System

A C++ project that manages parking slots for cars and bikes with automated billing and status tracking.

Project Overview

This project is a Smart Parking Management System built using C++. It helps automate vehicle parking in a limited number of slots, keeping track of cars and bikes, calculating parking fees based on duration, and displaying real-time parking status.

Features

  • Dynamic slot allocation for cars and bikes.
  • Prevents overbooking when slots are full.
  • Vehicle removal with automatic billing (₹5 per minute).
  • Displays current parking status with vehicle details.
  • Menu-driven console interface for easy usage.
  • Uses OOP concepts like classes, inheritance, and polymorphism.

Technologies Used

  • C++ — core programming language.
  • STL Map — for storing slot and vehicle details.
  • Classes & Inheritance — for Cars, Bikes, and generic Wheeler vehicles.
  • ctime — to calculate entry and exit duration.

How It Works

  1. Admin enters total number of available slots.
  2. User selects an option: Park, Remove, Display status, or Exit.
  3. When parking, the system stores vehicle type, number, and entry time.
  4. On removal, it calculates duration and charges ₹5 per minute.
  5. Status option displays all occupied slots with vehicle details.

Code Highlight

class slotbooking {
    int totalslots;
    map slots;

public:
    slotbooking(int limit) { totalslots = limit; }

    int parking(string num, string type) {
        if (filled()) {
            cout << "No slots available!";
            return -1;
        }
        int cnt = 1;
        while (slots.find(cnt) != slots.end()) cnt++;
        if (type == "Car") slots[cnt] = new Car(num);
        else if (type == "Bike") slots[cnt] = new Bike(num);
        cout << "Vehicle parked at slot: " << cnt;
        return cnt;
    }

    void remove(int cnt) {
        Wheeler* v = slots[cnt];
        double duration = difftime(time(0), v->entry)/60;
        double fee = 5 * duration;
        cout << "Pay: ₹" << fee;
        delete v;
        slots.erase(cnt);
    }
};
