Jarvis — Your Voice-Activated Virtual Assistant

A Python-powered assistant that listens, understands, and responds to your commands.

What is Jarvis?

Jarvis is a voice-activated assistant built in Python. It recognizes speech, executes tasks like opening websites, playing music, fetching news & weather, and even integrates with OpenAI for answering questions. Inspired by Iron Man’s J.A.R.V.I.S, it makes your computer interactive and smart.

Features

  • Voice activation with the wake word “Jarvis”
  • Open websites (Google, YouTube, IIIT Raichur)
  • Play songs from a custom music library
  • Fetch latest news using NewsAPI
  • Live weather updates using WeatherAPI
  • Tells current date and time
  • Handles general questions via OpenAI

Technologies Used

  • Python — main programming language
  • speech_recognition — converts speech to text
  • pyttsx3 — text-to-speech engine
  • requests — API integration for weather & news
  • webbrowser — opens websites
  • OpenAI API — intelligent responses

How It Works

  1. User says “Jarvis” to activate.
  2. System listens for commands via microphone.
  3. If the command matches known actions (like open Google, play music, or fetch weather), it executes directly.
  4. If it's a question, Jarvis forwards it to OpenAI for an intelligent reply.
  5. Responds back with both text and voice.
